Vorgang des Kaufs im Webshop:

Aus unserem Katalog können Sie den Inhalt Ihres Warenkorbes zusammenstellen. Auf dem ausführlichen Produktblatt des ausgewählten Produktes finden Sie jeweils den Button „In den Warenkorb“, vor dem Sie in eine kleine Rubrik die gewünschte Menge eintragen können. Wie Sie auch erkennen können, ist die Standardeinstellung dieser Rubrik 1 Stück. Nach Anklicken des Buttons „In den Warenkorb“ wird das ausgewählte Produkt in den Warenkorb gelegt und auf der Webseite der Inhalt des Warenkorbes angezeigt. Danach können Sie weitere Produkte kaufen, deren Menge in dem Warenkorb abändern bzw. Produkte aus dem Warenkorb herausnehmen, indem Sie einfach auf den Button „Entfernen“ (oder auf den „x“-Button) klicken. Wenn Sie die Mengen in dem Warenkorb abgeändert haben, klicken Sie auf den Button „Aktualisieren“ unter der Auflistung der Artikel; dadurch wird der Preis der Artikel im Warenkorb neu berechnet und die abgeänderten Mengen werden gespeichert. Nachdem Sie den Inhalt Ihres Warenkorbes zusammengestellt haben und die Artikel kaufen möchten, klicken Sie auf den Button „Bestellen“.

Nach Anklicken des Buttons „Bestellen“ kommen Sie auf die Seite der Auswahl der Lieferart. Das ist der 2. Schritt des Bestellvorganges. Die Schritte werden oben durch die eingekreisten Zahlen und den Text unter ihnen eindeutig angezeigt, damit Sie sehen können, wo Sie gerade im Bestellvorgang stehen. Wenn Sie von der Seite des Warenkorbes nicht auf die Seite der Lieferung weitergeleitet wurden, waren Sie vorher nicht in den Webshop eingeloggt. In diesem Fall wird eine Seite angezeigt, auf der Sie sich einloggen können, ohne dass Ihnen dabei der Inhalt des Warenkorbes verloren geht. Wenn Sie kein registrierter Anwender sind, können Sie sich – bei Erhaltung des Inhalts des Warenkorbes – auf der angezeigten Seite auch registrieren lassen. Nach erfolgreichem Einloggen oder Registrieren werden Sie zum Schritt der Auswahl der Lieferart weitergeleitet. Wählen Sie eindeutig die gewünschte Lieferart aus und klicken Sie anschließend auf den Button „Weiter“.

Sie müssen auf jeden Fall eine Lieferart auswählen, sonst lässt das System nicht zu, dass sie den Bestellvorgang fortsetzen. Auf der nächsten Seite wird der zu zahlende Betrag angezeigt und Angaben sind einzutragen. Hier sehen Sie den zu zahlenden Endbetrag und können auf dem Formular darunter die Rechnungs- und Lieferdaten angeben. Bei dem Feld Ihrer Kontaktdaten können Sie uns in der Rubrik Anmerkung auch eine kurze Nachricht senden. Nachdem Sie die Daten eingetragen haben, klicken Sie auf den Button „Weiter“. Nun sind Sie beim Schritt der Finalisierung der Bestellung angekommen. Hier können Sie die Daten nicht mehr abändern, aber alle Daten, die Sie angegeben haben, und alle bestellte Artikel, werden angezeigt. Prüfen Sie bitte die Daten sorgfältig! Wenn Sie einen Fehler entdecken, können Sie durch Anklicken des Buttons „Zurück“ unten zum entsprechenden Schritt zurückgehen und korrigieren. Nachdem Sie alle Daten in Ordnung gefunden haben, klicken Sie auf den Button „Bestellen“. Dann wird der der Text angezeigt, dass Ihre Bestellung erfolgreich war. Sie erhalten eine Mail als Bestätigung der Bestellung, in der alle Daten, die Sie angegeben haben, und alle bestellten Artikel, enthalten sind. Nach einer erfolgreichen Bestellung wird Ihr Warenkorb geleert. Nachdem Sie den Kauf abgeschlossen haben, können Sie sich oben im Feld „Ausloggen“ ausloggen.
